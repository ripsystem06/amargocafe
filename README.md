Amargo Café - Landing Page

Bienvenido al repositorio oficial del sitio web de Amargo Café, un espacio único en Ensenada, B.C. que combina el café de especialidad con la diversión de los juegos de mesa.

Descripción

Este proyecto es una Landing Page moderna y responsiva diseñada para convertir visitantes en clientes. Cuenta con un diseño "Dark Mode" elegante utilizando la paleta de colores de la marca, integración con redes sociales y accesos directos para reservas.

Puedes visitar el repositorio en: https://github.com/ripsystem06/amargocafe

Características Principales

Diseño Responsivo & Dark Mode: Adaptable a móviles, tablets y escritorio usando Tailwind CSS, con una paleta de colores personalizada (Tonos Café, Oro y Verde Bosque).

Galería de Reels Inmersiva: Integración de videos de Instagram utilizando una técnica avanzada de CSS Cropping para eliminar bordes blancos y distracciones, mostrando solo el video.

Menú Digital: Integración directa con el diseño de menú en Canva.

Botones de Acción (CTA):

Reservas y pedidos directos vía WhatsApp API.

Ubicación exacta mediante Google Maps.

Botón Flotante: Acceso rápido a WhatsApp persistente durante toda la navegación.

Seguridad Básica: Script de JavaScript (proteccion_codigo.js) para deshabilitar clic derecho y atajos de teclado de inspección.

Tecnologías Utilizadas

HTML5: Estructura semántica del sitio.

Tailwind CSS (CDN): Framework de utilidades para el estilizado rápido y moderno.

JavaScript Vanilla: Lógica de interacción y scripts de protección.

Lucide Icons: Librería de iconos vectoriales ligeros.

Estructura del Proyecto

amargocafe/
├── amargoimg/            # Carpeta de imágenes locales (si aplica)

Instalación y Uso Local

Para ver y editar este proyecto en tu computadora:

Clona el repositorio:

git clone [https://github.com/ripsystem06/amargocafe.git](https://github.com/ripsystem06/amargocafe.git)


Navega a la carpeta:

cd amargocafe


Ejecuta el sitio:
Simplemente haz doble clic en el archivo index.html para abrirlo en tu navegador web favorito.

Secciones del Sitio

Hero: Bienvenida impactante con botones para "Reservar/Pedir" y "Cómo llegar".

Experiencia: Descripción de los servicios (Café, Juegos, Coworking).
